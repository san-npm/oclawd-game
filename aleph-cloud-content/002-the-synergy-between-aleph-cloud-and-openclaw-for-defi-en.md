## Getting Started

### Quick Start Guide

```bash
# Initialize your DeFi project
npx openclaw init defi-protocol

# Set up Aleph Cloud integration
npx aleph-cloud connect

# Create your first DeFi contract template
npx openclaw create defi/lending-protocol

# Deploy to Aleph Cloud
npx aleph-cloud deploy --network ethereum
```
