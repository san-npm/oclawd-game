- Messages were accepted (202) but silently rejected on async signature verification
- Only 1 proof (test raffle 999) persisted; real draw proofs were lost
- Fix: one-line change in `lib/aleph-vrf.ts`, commit b113d22
- After fix: raffle #5 E2E test passed, proof confirmed on Aleph
- Source: `getVerificationBuffer()` in aleph-sdk-ts `packages/message/src/types/messages.ts`
