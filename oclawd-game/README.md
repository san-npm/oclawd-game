# Oclawd Space Strategy Game

A complete full-stack space strategy game built for Base testnet.

## Features

### Backend
- Express.js API with PostgreSQL database
- Station management system
- Resource trading
- Fleet operations
- Smart contract integration

### Frontend
- React-based UI
- Interactive galaxy view
- Fleet management dashboard
- Real-time marketplace

## Tech Stack
- Frontend: React, Tailwind CSS
- Backend: Express.js, Sequelize
- Database: PostgreSQL
- Blockchain: Base testnet
- Deployment: Docker, Kubernetes

## Quick Start

### Local Development
```bash
cd backend
npm install
npm run dev

cd ../frontend
npm install
npm run dev
```

### Docker Deployment
```bash
docker-compose up -d
```

### Kubernetes
```bash
kubectl apply -f k8s/
```
