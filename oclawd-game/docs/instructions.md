## Requirements

Before you start, you'll need:

1. **A Web3 Wallet** (MetaMask recommended)
2. **Base Sepolia Testnet ETH** (free from faucets)
3. **A modern browser** (Chrome, Firefox, Brave)

---

## Step 1: Set Up Your Wallet

### Install MetaMask

1. Go to [metamask.io](https://metamask.io)
2. Install the browser extension
3. Create a new wallet (save your seed phrase!)

### Add Base Sepolia Network

**Option A: Automatic**
- Click "Add Network" when prompted in-game

**Option B: Manual**
Add these settings in MetaMask:

| Setting | Value |
|---------|-------|
| Network Name | Base Sepolia |
| RPC URL | `https://sepolia.base.org` |
| Chain ID | 84532 |
| Currency | ETH |
| Explorer | `https://sepolia.basescan.org` |

---

## Step 2: Get Testnet ETH

You need free testnet ETH to play. Get some from these faucets:

1. **Coinbase Faucet** (Recommended)
   - Visit [faucet.quicknode.com/base/sepolia](https://faucet.quicknode.com/base/sepolia)
   - Connect wallet
   - Claim 0.1 ETH

2. **Alchemy Faucet**
   - Visit [sepoliafaucet.com](https://sepoliafaucet.com)
   - Enter your wallet address
   - Receive ETH

3. **Bware Labs**
   - Visit [bwarelabs.com/faucets/base-testnet](https://bwarelabs.com/faucets/base-testnet)
   - Claim daily

üí° **Tip:** Request from multiple faucets if one is empty!

## Step 3: Connect to Oclawd

1. Visit the game: **[oclawd.vercel.app](https://oclawd.vercel.app)** *(coming soon)*
2. Click **"Connect Wallet"**
3. Approve the MetaMask connection
4. Make sure you're on **Base Sepolia** network

---

## Step 4: Mint Your First Ship

1. Go to the **Fleet** page
2. Click **"Mint Ship"**
3. Select **Fighter** (cheapest at 0.001 ETH)
4. Confirm the transaction in MetaMask
5. Wait for confirmation (~2 seconds)
6. üéâ You now own your first ship!

---

## Step 5: Explore the Galaxy

1. Click **Galaxy** in the navigation
2. View the star map
3. Find asteroid fields (resource nodes)
4. Position your ship near resources

---

## Step 6: Start Mining

1. Select your ship in **Fleet**
2. Click **"Mine"** when near an asteroid
3. Confirm the transaction
4. Resources are added to your wallet
5. Check your balance on **Dashboard**

---

## Step 7: Trade on Marketplace

1. Go to **Marketplace**
2. Browse listings from other players
3. **To Buy:** Click item ‚Üí "Purchase" ‚Üí Confirm
4. **To Sell:** Click "List Item" ‚Üí Set price ‚Üí Confirm

## Troubleshooting

### "Transaction Failed"
- Check you have enough ETH for gas
- Make sure you're on Base Sepolia network
- Try increasing gas limit

### "Wallet Won't Connect"
- Refresh the page
- Check MetaMask is unlocked
- Try disconnecting and reconnecting

### "Can't See My Ships"
- Transactions take ~2 seconds to confirm
- Refresh the page
- Check the transaction on [Base Sepolia Explorer](https://sepolia.basescan.org)

### "Out of Testnet ETH"
- Visit faucets listed in Step 2
- Wait 24 hours and claim again
- Ask in Discord for community support

## Gameplay Tips

```
Day 1: Mint 2-3 Fighters
Day 2: Mine resources, sell on marketplace
Day 3: Build your first station
Day 4: Expand fleet, start exploring
Day 5+: PvP, compete on leaderboards
```

### Do's ‚úÖ

- Build a mining station early
- Check marketplace for deals
- Diversify your fleet
- Join the community

### Don'ts ‚ùå
