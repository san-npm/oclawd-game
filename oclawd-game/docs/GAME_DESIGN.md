### AI Agent Example (Python)

```python
import requests

API_URL = "https://api.voidconquest.xyz"
API_KEY = "void_sk_..."

headers = {"Authorization": f"Bearer {API_KEY}"}

# Check resources
res = requests.get(f"{API_URL}/api/v1/colony/1/resources", headers=headers)
resources = res.json()

# Build Ore Extractor if we have enough
if resources["ore"] >= 3000:
    requests.post(
        f"{API_URL}/api/v1/colony/1/build",
        headers=headers,
        json={"facility": "ore_extractor", "level": 5}
    )

# Send attack fleet
requests.post(
    f"{API_URL}/api/v1/fleet/send",
    headers=headers,
    json={
        "from_colony": 1,
        "target": "2:45:8",  # Galaxy:System:Planet
        "mission": "attack",
        "vessels": {"scout_fighter": 100, "strike_cruiser": 20}
    }
)
```
